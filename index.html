Project

You are tasked with writing a program to display the total bill to the user after adding tax and subtracting discount.
You need to create a function having the following parameters:

  1. n: The number of items purchased by the user.
  2. tax: By default, the value should be 0.2 (20%).
  3. total: By default, the value should be 0.
  4. items: The price of all the items purchased, passed as arbitrary arguments. Use recursion to calculate the total price of all items in the same function.
  5. discount, shipping_charge, coupon_code: Passed as arbitrary keyword arguments.

In the function, you need to do the following:

  • Find the total by using the concept of recursion.
  • Print the price after applying the discount (if any) to the total.
  • Print the price after applying the coupon code (if any) to the above discounted amount.
  • Print the price after applying the tax to the amount calculated in the above step.
  • Print the final payable amount after applying the shipping charge to the amount calculated in the above step.

Example:

If discount is 0.2, it means a 20% discount.
If coupon_code=50 that means 20 rupees off.
